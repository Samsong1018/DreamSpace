# DreamSpace Style Guide

## Inline Styles Policy
Inline styles are avoided. All global styles live in /css/styles.css.
Page-specific styles (if any) are placed in the <head> of that page only.
Inline styles are not used in this project.

## Color Palette
- Background: #0d0d1a (deep navy)
- Surface: #1a1a2e
- Accent: #7c6af7 (purple)
- Text: #e8e8f0
- Muted text: #9090aa

## Typography
- Font: Segoe UI / system-ui
- Base size: 1rem (16px)
- Hero: clamp(2rem, 5vw, 3.5rem)

## Design Direction
Dark atmospheric theme — deep indigo/navy palette evoking a dreamlike night aesthetic.
